# Create Mice App

Create mice.js apps with one command - a lightweight and modern web application generator.

## Quick Start

```bash
# Using npm
npm create mice-app my-app

# Using yarn
yarn create mice-app my-app

# Using pnpm
pnpm create mice-app my-app
```
